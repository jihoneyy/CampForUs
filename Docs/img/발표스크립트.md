#21p 시스템 구조도

```markdown
 먼저 시스템구조입니다.
 nginx를 앞에 배치하여
 proxy_pass를 통해
 기본주소로는 프론트엔드 프로젝트에 매핑하였고,
 api들어오면 장고 컨데이너, 그리고 search로 들어오면 크롤링 서버로 매핑을 하였습니다.

```

#22 소개

```markdown
 저희 캠퍼스는 크게 4가지 기능을 제공하고 있는데요,
 ```


```markdown

 ``` 
 
#25 캠핑장 상세정보


```markdown
캠핑장에 대해 알아보기

좀 더 간편하게 캠핑장에 대해 알아볼수 있도록 구성하였습니다.
 ```

#28 시연시나리오

```markdown
링크 클릭 후

로그인부터 해보겠습니다.


메인 페이지 입니다.
좋아요 순으로 가져온 캠핑장리스트를 맨 위에 보여주고,
그 밑으로 인기 태그별로 보여지도록 화면을 구성하였습니다.

(Najost Camp)클릭 
상세 페이지에서는 
이 캠핑장과 비슷한 캠핑장 10개를 추천해주고, 밑으로는 
캠핑장에 대한 정보와 지도가 보여집니다.

한 줄 리뷰는 로그인한 사용자가 자유롭게 캠핑장에 대한 
코멘트를 남길 수 있는 공간입니다.

(코멘트작성)
옆에 있는 휴지통을 누르게되면 삭제가 됩니다.
(코멘트 삭제)

네이버 검색 api를 사용하여 
해당 캠핑장에 대한 블로그 리뷰를 가져와 
이를 통해 다양한 사람들의 솔직한 정보를 얻을 수 있습니다.
(블로그 페이지 넘겨보기)

다음은 캠핑장 검색 기능입니다.

단어 또는 (태그검색클릭) 태그로도 검색을 할 수 있습니다.
(단어클릭 후, 수영장 검색) 단어 검색을 통해

(태그로 검색 클릭)
태그 검색을 통해서는 사용자의 성향에 맞게
원하는 종류의 캠핑장을 검색 해 볼 수 있습니다.
(숲에 위치한, 글램핑 즐기기 좋은 클릭)

(마이페이지 클릭)
마이페이지를 통하여 내가 좋아요 누른 캠핑장들과
내가 작성 한 댓글을 확인 할 수 있습니다.

이상으로 시연을 마치겠습니다.
```
